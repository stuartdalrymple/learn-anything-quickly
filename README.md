# learn-anything-quickly
Learn 80% of the content in 20% of the time. A poor man's LMS for Claude AI.

Quick install:
1. Start a new Claude Project
2. Copy contents of OS file into 'Project instructions'
3. Upload 'brain template' file into Project Files
4. Use this prompt with Claude: "Run the interview to build my first brain file"
5. Upload the generated file into Project Files
6. Ask Claude to start a new session whenever you're ready to learn

Full install instructions at: stuartdalrymple.com/learn-anything-quickly.html
